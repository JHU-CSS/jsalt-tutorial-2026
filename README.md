# JSALT — Responsible AI and Evaluation (Part 1)

Hands-on labs covering predictive fairness, fighting-words analysis, and text representations.

Notebooks (in each folder below): `lab1_predictive_compas.ipynb`, `lab2_fighting_words.ipynb`, `lab3_representations.ipynb`.

## Pick a variant

- **[`groq/`](groq/)** — labs using the **Groq API** for LLM calls. Start here.
- **[`local_llm_call/`](local_llm_call/)** — same labs with a **local LLM backup** (Ollama or vLLM) that takes over when Groq is unavailable or no key is set.

Each folder has its own `README.md` with setup, environment, and API-key instructions.
